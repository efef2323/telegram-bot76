ИНСТРУКЦИЯ ДЛЯ ДЕНИСА:

ШАГ 1: Подготовка
1. Создай новую папку на Рабочем столе с названием "TradingBot"
2. Скопируй все 4 файла в эту папку

ШАГ 2: Установка библиотек
1. Дважды кликни на файл "install_requirements.bat"
2. Дождись установки всех библиотек

ШАГ 3: Запуск бота
1. Дважды кликни на файл "start_bot.bat"
2. Бот запустится автоматически

ПУТИ К ФАЙЛАМ:
- Скриншоты сохраняются в папку: TradingBot\screenshots\
- Логи сохраняются в папку: TradingBot\logs\

ВАЖНО:
1. Бот использует твои данные (логин и пароль)
2. Перед запуском закрой все окна Chrome
3. Если бот не может найти кнопки - он сделает скриншот
4. Все скриншоты сохраняются в папке screenshots

ПРОБЛЕМЫ И РЕШЕНИЯ:
1. Если ошибка "Chrome not found" - установи Google Chrome
2. Если ошибка "pip not found" - установи Python с сайта python.org
3. Если бот не видит кнопки - проверь скриншоты в папке

КОНТАКТЫ:
Если что-то не работает - напиши какие ошибки видишь в консоли
